Computer Science (CSE) Course Descriptions

CSE 101: Introduction to Computers

An introduction to computers & computing for non-majors. Covers problem solving, basic programming concepts, and the societal impact of computing. No prior programming experience required. 3 credits.

CSE 114: Introduction to Object-Oriented Programming

Introduces the fundamental concepts of object-oriented programming in Java: classes, objects, inheritance, and polymorphism. Intended as the first course for computer science majors. 4 credits.

CSE 214: Data Structures

A study of data structures and their algorithms: lists, stacks, queues, trees, hashing, and graphs, with an emphasis on implementation and asymptotic analysis. Prerequisite: CSE 114. 4 credits.

CSE 220: Systems Fundamentals I

An introduction to the low-level execution environment: data representation, assembly language, the memory hierarchy, and the relationship between high-level code and the underlying hardware. Prerequisite: CSE 214. 3 credits.
